## 商品数据相关的SP-API接口
商品数据主要是通过SP-API的ReportAPI来获取。对应的接口文档地址是 [Reports V2021-06-30](https://developer-docs.amazon/sp-api/reference/reports-v2021-06-30) 。
使用的报告类型是：`GET_MERCHANT_LISTINGS_ALL_DATA` 隶属于Inventory分类。 对应的文档地址是：[All Listings Report](https://developer-docs.amazon/sp-api/docs/report-type-values-inventory#all-listings-report)
